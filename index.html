HTML+CSS & Marks+Channels

The above chart is a compound bubble and pie chart. The data type for the above figure is a quantitative attribute that represents the items. There is quantitative continuous data on either axis. The data encompassed in the pie charts is categorical, distributed into 3 categories. The size of the angles in the data are out of 100 for the angles. The marks for the above chart are the containment areas between subcategories that indicate the amount of the third value for the bubbles and these bubbles are also used as points. There are a lot of different channels that are used to represent the data. The position is used for x and y axes for the attribute items, the tilt creates and represents the angle and size of the bubble charts, the size for the area of the bubbles for the 3 categories as mentioned above and the color for the different categories in the separate pie charts in the chart shown above.
